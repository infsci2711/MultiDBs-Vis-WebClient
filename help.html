How to show data in charts?


1- Select Canvas or create Canvas.
2- Select Story or add story.
3- Select Column(s) from the story.
4- Select already created chart or create new chart.
4- In creating new chart, select chart type and aggregate function if it is required.
5- Click on Create chart.

Click on create button and fill Canvas name in the Popup window.


Select Canvas and then click on Open button.


Click on Story if it is collapsed, Select Story and then click on Open button.


Select existing chart by selecting it and the clicking on Open button or click on create button to create new chart.


To create chart, select the required properties like: columns and aggregate functions.
